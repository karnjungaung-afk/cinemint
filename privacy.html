🔒 Privacy Policy

นโยบายความเป็นส่วนตัว

CineMint ให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้ทุกคน นโยบายนี้อธิบายว่าเราเก็บข้อมูลอะไร ใช้อย่างไร และคุณมีสิทธิ์อะไรบ้าง

อัปเดตล่าสุด: 24 มิถุนายน 2026  •  มีผลบังคับ: 24 มิถุนายน 2026

1. ข้อมูลที่เราเก็บ

CineMint เป็นเว็บไซต์ฝั่ง Client-side ล้วนๆ ที่ทำงานในเบราว์เซอร์ของคุณ เราไม่มีเซิร์ฟเวอร์ที่รับและจัดเก็บข้อมูลส่วนตัวของผู้ใช้ อย่างไรก็ตาม ข้อมูลบางส่วนอาจถูกเก็บในอุปกรณ์ของคุณเอง ได้แก่

  • localStorage: เก็บการตั้งค่า TMDB API Key (ถ้าคุณกรอก) และแคชข้อมูลภาพยนตร์เพื่อลดการเรียก API ซ้ำ ข้อมูลเหล่านี้อยู่บนอุปกรณ์ของคุณเท่านั้น ไม่ถูกส่งมาที่เรา
  • แบบฟอร์มติดต่อ: หากคุณกรอกฟอร์มติดต่อ ข้อมูลจะถูกประมวลผลใน JavaScript เพื่อตรวจสอบรูปแบบเท่านั้น ไม่มีการส่งข้อมูลไปที่เซิร์ฟเวอร์ใดๆ ในเวอร์ชันสาธิตนี้

2. บริการของบุคคลที่สาม

CineMint ใช้บริการของบุคคลที่สามดังต่อไปนี้ ซึ่งมีนโยบายความเป็นส่วนตัวของตนเอง

  • The Movie Database (TMDB): ใช้สำหรับดึงข้อมูลโปสเตอร์และข้อมูลภาพยนตร์ เมื่อเบราว์เซอร์โหลดรูปภาพจาก image.tmdb.org TMDB อาจบันทึก IP address ของคุณตามนโยบายของพวกเขา
  • YouTube (Google): ใช้สำหรับฝังตัวอย่างหนัง เมื่อโหลด iframe ของ YouTube, Google อาจวาง cookie ในเบราว์เซอร์ตามนโยบายของ Google
  • Google Fonts: ใช้โหลดฟอนต์ Fraunces และ Inter จาก Google Fonts CDN ซึ่ง Google อาจบันทึก request log
  • Font Awesome: ใช้โหลด icon จาก Cloudflare CDN

3. คุกกี้และ Local Storage

CineMint ไม่วาง คุกกี้ติดตาม (Tracking Cookies) ของตัวเองเลย ข้อมูลใน localStorage ที่เราสร้างขึ้นมีไว้เพื่อการทำงานของเว็บไซต์เท่านั้น ได้แก่ แคช TMDB API และการตั้งค่า

คุณสามารถลบข้อมูล localStorage ได้ตลอดเวลาผ่าน Developer Tools ของเบราว์เซอร์ (F12 → Application → Local Storage)

4. ความปลอดภัยของข้อมูล

เนื่องจาก CineMint ไม่มีเซิร์ฟเวอร์รับข้อมูล ความเสี่ยงจากการรั่วไหลของข้อมูลจากฝั่งเราจึงมีน้อยมาก อย่างไรก็ตาม เราแนะนำให้

  • ไม่แชร์ TMDB API Key ของคุณกับผู้อื่น
  • ล้าง localStorage เป็นครั้งคราว โดยเฉพาะเมื่อใช้คอมพิวเตอร์สาธารณะ
  • ใช้เบราว์เซอร์ที่อัปเดตเป็นเวอร์ชันล่าสุดอยู่เสมอ

5. ผู้ใช้ที่เป็นผู้เยาว์

CineMint ไม่ได้ออกแบบมาเพื่อเก็บข้อมูลของผู้เยาว์อายุต่ำกว่า 13 ปีโดยเฉพาะ เนื้อหาภาพยนตร์บางเรื่องในฐานข้อมูลของเราอาจมีความรุนแรงหรือเนื้อหาสำหรับผู้ใหญ่ ผู้ปกครองควรดูแลการใช้งานของบุตรหลานตามความเหมาะสม

6. สิทธิ์ของคุณ

เนื่องจากเราไม่ได้เก็บข้อมูลส่วนตัวของคุณบนเซิร์ฟเวอร์ คุณจึงมีสิทธิ์ควบคุมข้อมูลของตนเองได้อย่างสมบูรณ์

  • สิทธิ์เข้าถึง: ดูข้อมูลที่เก็บใน localStorage ผ่าน Developer Tools
  • สิทธิ์ลบ: ลบข้อมูลทั้งหมดได้ผ่าน Clear Site Data ในเบราว์เซอร์
  • สิทธิ์คัดค้าน: ปิด YouTube iframe หรือบล็อก Google Fonts ผ่านส่วนขยายของเบราว์เซอร์

7. การเปลี่ยนแปลงนโยบาย

เราอาจอัปเดตนโยบายนี้เป็นครั้งคราว การเปลี่ยนแปลงจะแสดงวันที่อัปเดตที่ด้านบนของหน้านี้ การใช้งานเว็บไซต์ต่อหลังจากการเปลี่ยนแปลงถือว่าคุณยอมรับนโยบายใหม่
